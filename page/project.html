PROJECT

〇Background

　DNAはその相補性に基づいて凝集することで、 ハイドロゲルを形成できることが知られる。 例えば、粘着末端を持つ二本鎖のリンカーを利用し Y-モチーフDNAを接続させることでゲル化させる方法などが知られる。 こうして形成されるゲルにはDNAを材料とするメリットがいくつかある。 ひとつめは、ゲルを構成するDNAに特定の遺伝子を組み込むことが出来る点である。 こうすることでタンパク質をハイドロゲル内で無細胞合成することが可能である。 また、ゲルを形成するDNAにアプタマー配列を組み込むことで特定の物質や細胞などを特異的にゲル内に取り込むことができる。 これを応用すれば、特定の細胞のみを取り分けたり、薬剤をDNAゲル構造で保護しながらデリバリーすることが期待される。
　同時に、DNAを材料としたインテリジェンスなツールは数多く開発されている。 アンドゲートやノットゲートといった論理回路はもちろん、 それらをトランスデューサーとして組み込んだ分子ロボットや、 複数組み合わせて構築されたコンピューティングネットワークなども知られる。 これは、DNAの相補的な結合やそれに伴う二次構造・三次構造の形成、 あるいはNucleaseや制限酵素などのツールの豊富さによって成り立っている。

〇Our Object

　私たちのプロジェクトは、DNAツールのインテリジェンシーを利用してゲル化を制御することで、 特異的・局所的にDNAハイドロゲルのメリットを提供するデバイスの設計である。 具体的には、インプットとなるDNAを受け取ったときに自己凝集をして濃度を上げることでゲルを形成するような群体デバイスだ。
　そこで私たちは、2次元構造の形成とゲル形成どちらにも実績がある、RCA産物の長鎖DNAに目をつけた。 これらの実績を再現し、さらに鎖置換の要素を加えることで２つの実績を関連づけることで、私たち独自のデバイスをデザインしました。 このデバイスの挙動を評価することが私たちのプロジェクトの目的です。
